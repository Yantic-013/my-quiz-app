# 开发执行步骤

## 阶段一：基础骨架 + 题库导入
1. 创建项目目录结构（dev-log/、docs/、templates/）
2. 创建 CLAUDE.md、标准文档
3. 创建 quiz-app.html：HTML 结构 + CSS 主题 + 导航框架
4. 实现题库导入：SheetJS CDN → 解析 Excel → 存 localStorage → 显示导入结果
5. 创建 Excel 模板文件
6. 记录开发日志

## 阶段二：章节练习
1. 章节列表页：提取章节 + 题目数量统计 + 选择章节
2. 答题卡片：题目显示 + 选项随机排列 + 选项自适应布局
3. 答题逻辑：单选点击提交 / 多选勾选提交 + 对错判断 + 解析展示
4. 章节统计：完成正确率 + 跳转下一题

## 阶段三：错题集
1. 错题收录：答题错误时自动写入 errorBook
2. 错题集页：列表展示 + 按章节筛选 + 错误次数显示
3. 错题重练：进入练习模式，答对可选移除

## 阶段四：模拟考试
1. 配置页：章节多选 + 单选/多选数量输入 + 可抽数量实时显示
2. 抽题算法：从所选章节按题型随机抽取
3. 考试模式：答题流程 + 结果统计（单选/多选分计 + 章节得分）

## 阶段五：背题模式
1. 章节练习中增加"背题/刷题"切换按钮
2. 背题卡组件：直接展示答案，翻页浏览
3. 确保不触发错题收录逻辑

## 阶段六：数据管理 + 收尾
1. 导出题库/错题集为 JSON 下载
2. 清空数据功能（带确认弹窗）
3. 全流程测试，修复边界问题
4. 更新开发日志
