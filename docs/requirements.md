# 功能需求规格

## 1. 题库导入
- 支持导入 Excel (.xlsx) 文件
- Excel 列结构：章节 | 题型 | 题目 | 选项A~N | 正确答案 | 解析
- 选项列至少 4 列（A-D），可扩展至更多列（E、F...），程序自适应识别
- 题型：单选 / 多选
- 正确答案格式：单选填单个字母（如 `A`），多选填逗号分隔字母（如 `A,C,E`）
- 支持增量导入（追加新题目，已有题目不重复覆盖）
- 导入数据存入 localStorage，持久化保存

## 2. 章节练习
- 按章节展示题目列表，显示每章题目数（单选/多选分别统计）
- 答题时选项顺序每次随机打乱
- 单选题：点击单个选项即提交，立即显示对错 + 解析
- 多选题：点击选项选中，点击提交按钮确认，显示对错 + 解析
- 答错的题自动收录到错题集
- 章节完成后显示正确率统计

## 3. 模拟考试
- 用户选择考核章节（可多选）
- 自主设定单选题数和多选题目数
- 系统从所选章节中按题型随机抽取题目
- 抽题不足时提示用户调整数量
- 考试结束显示总成绩（单选/多选分开计分）和各章节得分

## 4. 错题集
- 自动收集所有错题，记录错误次数
- 按章节筛选查看
- 支持错题重练（答对后可选从错题集移除）
- 显示每题错误次数统计

## 5. 背题模式
- 进入章节后可切换为背题模式
- 直接显示题目和正确答案，无答题交互
- 不产生错题记录，不记录答题历史
- 可前后翻页浏览

## 6. 数据管理
- 导出题库为 JSON 文件（备份）
- 清空题库
- 导出/清空错题集
- 题库总览（各章节单选/多选数量统计）
